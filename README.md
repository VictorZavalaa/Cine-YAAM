# Cine-YAAM 🎬

Un proyecto web desarrollado en **Laravel** diseñado para administrar un catálogo de películas y usuarios, con sistema de autenticación completo y diseño responsivo.

## ✨ Características Principales

* **Autenticación y Autorización:**
  * Registro, inicio de sesión y cierre de sesión.
  * Sistema de roles: Usuarios regulares y Administradores.
  * Protección de rutas mendiante Middlewares (`auth`, `admin`).
* **Gestión de Películas (CRUD):**
  * Control total sobre las películas: Creación, lectura, actualización y eliminación de registros.
  * Campos detallados: ID Externo, Título, Resumen (Overview), Fecha de Estreno y Ruta del Póster.
* **Gestión de Usuarios (CRUD):**
  * Panel administrativo para gestionar la tabla principal de usuarios (Nombre, Email, Teléfono, Rol, etc).
* **Alertas Globales e Interactivas:**
  * Notificaciones emergentes dinámicas (Verdes de `Éxito`, Rojas de `Peligro/Eliminación`) y control automático de **Errores de Validación**.
  * Auto-desvanecimiento con animaciones en JavaScript.
* **Diseño e Interfaz:**
  * Uso de **Bootstrap 5** para maquetación e interfaces responsivas.
  * Íconos modernos implementados vía **FontAwesome**.
